PICTURE THE FLAG - Les drapeaux HTML

Objectif de Picture the Flag

L'objectif de ce challenge est de reproduire chacun des drapeaux ci-dessous à l'aide de tableaux HTML. Chaque tableau devra comporter le minimum de cases, c'est-à-dire qu'il faut fusionner (colspan et rowspan) les cases autant que possible.

Moins, c'est mieux
Plus (+) de fusions, moins (-) de cases...

Quelques précisions, à toutes fins utiles...

Le code proposé est étudié pour vous faire manipuler les tableaux HTML. En conséquence, il n'est pas "optimisé" comme il pourrait/devrait l'être dans un cadre professionnel.

Cette page est destinée aux développeurs, qui travaillent majoritairement sur des ordinateurs de bureau. (Donc peu de media queries dans le code CSS)

Cette page n'est pas optimisée en termes d'accessibilité numérique.
